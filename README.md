# 2D Game (MonoGame)

## Screenshots

This project is a 2D game developed using the MonoGame framework and C#.

The project implements object collision detection and sprite animation systems, forming the core mechanics of the game.

It also demonstrates the use of object-oriented programming principles along with selected design patterns to create a structured and maintainable codebase.

Additionally, the game includes basic input handling and entity management, allowing interaction within the game world.

This project serves as a platform for experimenting with game development concepts and building a deeper understanding of how 2D games are structured and implemented.
